HitMe
=====
A Location Based Information Exchange


How It Works
============ 

Create A Profile
	- Full Name
	- UserName
	- Email Address
	- Phone Number
	- Profile Picture 

Nearby Feed
	- List of people nearby 
	- In order of proximity
	- Person Profile
		- Name of person
		- Picture
